# Building local images with Vite

## Compatibility

- vite v5
- node 20+

## Installation

::: code-group

```bash [npm]
npm install @responsive-image/vite-plugin
```

```bash [yarn]
yarn add @responsive-image/vite-plugin
```

```bash [pnpm]
pnpm add @responsive-image/vite-plugin
```

:::
